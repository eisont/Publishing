프로그래밍이란 무엇인가?

사람이 없어도 알아서 운전하는 자율 주행 자동차나 바둑 기사를 이긴 인공지능 알파고 뉴스를 들어 본 적이 있나요? 이런 뉴스를 들으면 컴퓨터로 못할 일이 없을 것 같습니다. 하지만 사실 컴퓨터는 전원이 없으면 아무것도 못하는 전자 장치입니다. 컴퓨터가 이렇게 똑똑한 이유는 컴퓨터 안에 담겨 있는 프로그램 덕분입니다.

프로그래밍이란?

프로그램 만드는 과정

조금 더 어렵게 말하면 사람이 원하는 대로 컴퓨터가 작동할 수 있도록 컴퓨터 언어로 명령어를 나열하는 행위

웹 프로그래밍이란?

웹에 관련된 프로그램을 만드는 행위
웹 프로그래밍은 웹 브라우저와 관련된 프로그램을 작성하는 것을 가르킵니다.

서버란?

사용자의 검색 요청을 처리해 알맞은 정보를 전달해주는 컴퓨터

클라이언트란?

웹 브라우저를 보여주는 pc나 노트북, 모바일 기기
좁은 의미로 웹 브라우저
