자바스크립트로 무엇을 할 수 있을까?

웹사이트를 동적으로 만들 수 있습니다.

css로도 동적으로 만들수 있지 않나요?

결론부터 말하자면 한계가 있습니다. 요즘 사이트에서 보시는 모션들을 구현하기에는 css만으로 부족합니다.

HTML, CSS, JS에 역할을 간단히 알아봐요.

사람의 몸으로 예를 들어 봅시다.

HTML === 우리 몸을 지탱하는 뼈 CSS === 보이는 외관을 담당하는 살 JS === 인간의 몸이 움직이기 위해 필요한 근육이나 신경들

JS의 특징?

  1. 자바스크립트는 웹 브라우저에서 사용하기 위해 만든 언어입니다.
    그렇기에 모든 웹 브라우저에서 작동합니다. 다만 자바스크립트의 최신 문법을 웹 므라우저에서 즉시 사용할 수 있는 것은 아닙니다.

  2. 자바스크립트만으로 애플리케이션도 만들수 있습니다.
  3. 다양한 API를 사용해서 보다 정보가 풍부한 사이트를 만들 수 있습니다.

    API란?
    어떤 정보(데이터)를 다른 사람이 손쉽게 가져갈 수 있도록 미리 준비한 체계

  4. 다양한 라이브러리와 프레임워크를 사용할 수 있다.

    자바스크립트 라이브러리란?

    자바스크립트로 미리 구현해 놓은 기능을 묶어 놓은 것

    예를 들면 === 제이쿼리 jQuery

    자바스크립트 프레임워크란?

    프레임워크에서 기본으로 제공하는 소스를 수정하거나 추가하는 방법으로 웹 프로그램을 만들 수 있게 미리 준비한 일종의 틀

    예를 들면 앵귤러Angular와 뷰Vue가 있다.

클라이언트란?

웹 브라우저를 보여주는 pc나 노트북, 모바일 기기
좁은 의미로 웹 브라우저
